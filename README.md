# Loan Calculator Plugin (WordPress)

A modern and interactive WordPress plugin that allows users to calculate loan repayments in real time with both visual (pie chart) and written breakdowns. The calculator updates instantly as users adjust values using sliders or input fields.

---

## 🚀 Features

### 📊 Interactive Loan Calculator
- Calculate loan repayment instantly
- Real-time updates on every input change
- Supports:
  - Loan Amount
  - Interest Rate
  - Loan Term (Years)

### 🎛️ User-Friendly Controls
- Slider inputs for smooth value adjustment
- Manual input fields for precise entry
- Fully synchronized inputs (slider ↔ input field)

### 📈 Visual Output
- Dynamic **pie chart breakdown** showing:
  - Principal amount
  - Total interest
- Written summary of:
  - Monthly payment
  - Total repayment
  - Total interest paid

### ⚡ Real-Time Experience
- No page reload required
- Instant recalculation on every change
- Smooth and responsive UI

---

## 🧮 How It Works

1. User enters loan amount, interest rate, and loan duration
2. User adjusts values using sliders or input fields
3. Plugin automatically calculates:
   - Monthly payment
   - Total repayment
   - Total interest
4. Results are displayed in:
   - Pie chart visualization
   - Text-based summary
